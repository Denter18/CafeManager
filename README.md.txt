# CafeManager

Это проект Windows Forms приложения на C#, предназначенный для управления кафе.

## Основные возможности
- Авторизация пользователей
- Управление меню
- Работа с заказами
- Формирование отчётов
- Работа с базой данных SQLite

## Требования
- .NET Framework 4.7.2
- Visual Studio (рекомендуется 2019 или новее)

## Запуск
проект готов к запуску, и всё, что нужно — просто запустить CafeManager.exe.
